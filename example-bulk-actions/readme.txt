== Description ==

This plugin is an example for new WordPress 4.7 feature, bulk actions filter. Install it and go to the Pages menu and see new item in bulk menu.

Please note, that you need WordPress 4.7 or later to use this plugin.

== Installation ==

1. Unzip `example-bulk-actions` folder
1. Upload `example-bulk-actions` folder to your `/wp-content/plugins/` folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's all folks, have fun :) Read source code and work with it!
